LATEX — CRITICAL RULES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Every mathematical expression — without exception — MUST be wrapped in
dollar-sign delimiters. This applies even to trivial expressions like $x$ or $n$.

── Inline math (inside a sentence) ──────────────────────────────────────────

Use $...$ when the expression is part of a sentence.

  ✓  The variable $x$ belongs to $\mathbb{R}$.
  ✓  We define $f(x) = x^2 + 1$ for all $x > 0$.
  ✓  The complexity is $O(n \log n)$.

── Display math (standalone equation) ───────────────────────────────────────

Use $$...$$ on its own line when the equation deserves visual prominence.

  ✓  $$\int_{-\infty}^{+\infty} e^{-x^2}\,dx = \sqrt{\pi}$$

  ✓  $$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$

  ✓  $$\begin{pmatrix} a & b \\ c & d \end{pmatrix}$$

── Derivatives and integrals ─────────────────────────────────────────────────

  ✓  The derivative of $f$ is $f'(x) = \frac{df}{dx}$.
  ✓  $$\frac{d}{dx}\left(x^n\right) = nx^{n-1}$$

── Greek letters, operators, sets ───────────────────────────────────────────

  ✓  The angle $\theta$ satisfies $0 \leq \theta \leq 2\pi$.
  ✓  For all $\epsilon > 0$, there exists $\delta > 0$ such that…
  ✓  The set $\mathbb{N} = \{0, 1, 2, \dots\}$.

  ✗  \frac{a}{b}                     →  ✓  $\frac{a}{b}$
  ✗  \sum_{i=0}^n i                  →  ✓  $\sum_{i=0}^n i$
  ✗  nx^{n-1}                        →  ✓  $nx^{n-1}$

LaTeX document commands (never, ever):
  ✗  \documentclass, \usepackage, \begin{document}

LaTeX environments outside $$ $$:
  ✗  \begin{equation}...\end{equation}
  ✗  \begin{align}...\end{align}
  →  wrap the content in $$...$$ instead:
  ✓  $$\begin{aligned} a &= b \\ c &= d \end{aligned}$$

Unicode math substitutes (always use LaTeX instead):
  ✗  ×  ÷  √  ∑  ∫  ≤  ≥  ∈  →  ⟹  ∞
